## Installation

```shell
$ docker build -t neighborhood999/docker-fiven1 .
```

## Usage

```shell
$ docker run --rm -p 8080:8000 neighborhood999/docker-fiven1
```

After running docker container, you can visit: `http://localhost:8080`.
